# 单词大闯关

一款适合小学 3-6 年级孩子使用的离线单词闯关网页应用。

## 在线体验

直接用浏览器打开 `index.html` 即可使用，无需安装、无需网络。

## 文件说明

- `index.html` —— 单词大闯关主应用
- `phonics.html` —— 自然拼读小课堂（独立应用，框架已搭建，待填充真实课程资料）
- `CHANGELOG.md` —— 版本更新日志
- `DEVELOPMENT.md` —— 完整开发流程文档

## 主要功能

- **6 种练习模式**
  - 中文 → 英文拼写
  - 英文 → 中文释义
  - 中文选英文（卡片）
  - 英文选中文（卡片）
  - 听音辨词
  - 看图片选词

- **作业导入**：粘贴老师布置的单词列表，生成自定义词包
- **答后学习卡**：答对后显示音标、自然拼读拆分、例句
- **错题本**：自动记录错题，支持导出/导入/再练
- **学习统计**：练习单词数、已掌握单词、总正确率
- **打卡系统**：每日打卡、连续打卡天数
- **学习设置**：音效开关、朗读语速、每轮题量
- **键盘支持**：卡片模式按 1-4 选择，ESC 返回首页
- **自然拼读**：独立的 `phonics.html` 应用，按级别学习发音

## 使用说明

1. 下载或克隆本仓库
2. 双击 `index.html` 打开「单词大闯关」
3. 双击 `phonics.html` 打开「自然拼读小课堂」
4. 选择词包/级别和模式，开始学习

## 自然拼读应用

`phonics.html` 是独立的自然拼读学习应用，当前为框架版本，包含：

- 5 个级别：字母发音、短元音、长元音、辅音连缀、字母组合
- 4 种学习模式：学习模式、听音辨音、拼读练习、听音写词
- 进度条和发音朗读

后续将根据牛津自然拼读（Oxford Phonics World）和 Sam 教学法填充真实课程数据。

## 技术说明

- 单文件 HTML，内联 CSS 和 JavaScript
- 使用浏览器 Web Speech API 进行语音朗读
- 使用 localStorage 保存错题、打卡、设置和学习统计
- 完全离线可用，无外部依赖
