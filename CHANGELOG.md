# 单词大闯关 更新日志

## [1.0.0] - 2026-06-28

### 已实现功能
- 内置小学 3-6 年级暑假复习词包
- 6 种练习模式：中文→英文拼写、英文→中文释义、中文选英文、英文选中文、听音辨词、看图片选词
- 卡片模式：4 选 1 答题，自动判定
- 动画效果：小狐狸 mascot、星星爆炸、进度条、奖杯、连击显示
- 音效反馈：答对/答错/胜利音效
- 朗读功能：单词朗读、例句朗读，使用 Web Speech API
- 自然拼读：音标显示、字母拆分高亮
- 例句展示：每词配套英文例句及中文翻译
- 评分系统：基础 10 分 + 连击加分 + 全对奖励
- 错题本：自动记录、支持导出/导入/清空
- 打卡系统：localStorage 记录每日打卡、连续打卡天数
- 作业导入：粘贴单词列表生成自定义词包
- 答题页可中途返回首页

### 已知问题
- 自定义词包刷新后丢失
- 缺少设置面板（无法关闭音效、调节语速）
- 卡片模式不支持键盘选择
- 缺少学习统计（每个单词的掌握程度）
- 导入解析对复杂格式不够健壮

## [1.1.0] - 2026-06-28

### 新增
- 自定义词包持久化：导入的作业词包保存到 localStorage，刷新不丢失
- 设置面板：可开关音效、调节朗读语速、选择每轮题量（5/6/7/8/全部）
- 键盘支持：卡片模式按 1-4 选择答案，按 ESC 返回首页
- 学习统计：记录每个单词的练习次数、正确次数、最近掌握时间
- 掌握度标记：连续答对 3 次的单词自动标记为「已掌握」，可降低出题频率
- 导入增强：支持 `#` 注释、逗号/空格/制表符分隔、自动忽略空行
- 错题本去重优化：导入时自动合并重复错题

### 优化
- 重构状态管理：使用 `appState` 对象替代零散全局变量
- 改进自动播放策略：听音模式在首次用户交互后才自动朗读
- 移动端适配：设置面板和统计页针对小屏幕优化

### 修复
- 修复朗读重叠问题：切换题目时取消进行中的语音
- 修复自定义词包在看图片模式下无图标时显示异常
